Αρχικά, όχι, δεν είναι μία ανάρτηση για τον Γ' Παγκόσμιο Πόλεμο, ούτε ξεκινήσαμε στο Aftermaths να έχουμε ανησυχίες για τέτοια ζητήματα, τουλάχιστον όχι ακόμα. Αυτό που θα μας απασχολήσει εδώ είναι μία σχεδόν κλασσική απορία πάνω στη σχολική γεωμετρία.

Όλες και όλοι μας γνωρίζουμε τα περιβόητα κριτήρια ισότητας τριγώνων: ΠΠΠ, ΠΓΠ και ΓΠΓ ή, λίγο πληρέστερα: Πλευρά-Πλευρά-Πλευρά, Πλευρά-Γωνία-Πλευρά και Γωνία-Πλευρά-Γωνία. Η αλήθεια είναι ότι χρησιμοποιώντας αυτές τις, αν μη τι άλλο, βολικές συντομογραφίες, χάνουμε πολλές φορές τα λεπτά σημεία που εμφανίζονται στα δύο τελευταία κριτήρια. Για την ακρίβεια, σε ό,τι αφορά το περιβόητο ΠΓΠ, η ακριβής διατύπωση είναι η εξής:

Δύο τρίγωνα που έχουν δύο αντίστοιχες πλευρές και τις περιεχόμενες σε αυτές γωνίες ίσες μία προς μία είναι ίσα.

ΠΓΠ

Η λέξη κλειδί εδώ ο όρος περιεχόμενες. Δε μας απασχολεί να έχουμε απλώς δύο πλευρές και μία γωνία ίσες, αλλά η γωνία αυτή να βρίσκεται «ανάμεσα» στις δύο πλευρές που είναι ίσες στα δύο τρίγωνα. Με άλλα λόγια οι ίσες πλευρές των δύο τριγώνων να αποτελούν ταυτόχρονα και πλευρές των δύο γωνιών. Ωστόσο, είναι αυτή η υπόθεση απαραίτητη;

Δύο πλευρές και μία γωνία

Συνήθως, σε αυτό το σημείο, είτε μιλάμε για Γ' γυμνασίου είτε για Α' λυκείου, κάποιο χέρι θα σηκωθεί μέσα στην τάξη και θα ακουστεί ακολουθήσει ο εξής διάλογος:

[Χέρι] Μα γιατί να είναι η περιεχόμενη, αφού ισχύει όποια κι αν είναι η γωνία;
[Εκπαιδευτικός] Πώς μπορείς να το στηρίξεις αυτό;
[Χέρι] Να, έκανα τόσα τρίγωνα και δε μπόρεσα να βρω κάποια που να μην ισχύει αυτό, όποια κι αν ήταν η γωνία.

Η αλήθεια είναι ότι πράγματι, αν κάνει κανείς κατασκευές με το χέρι χωρίς την απαραίτητη αυστηρότητα είναι δύσκολο να βρει μία περίπτωση στην οποία να έχουμε δύο τρίγωνα με δύο πλευρές και μία γωνία ίσες τα οποία ωστόσο να μην είναι ίσα. Ας ξεκινήσουμε, λοιπόν, βήμα-βήμα, να κατασκευάσουμε ένα τέτοιο αντιπαράδειγμα - έτσι θα αποδείξουμε και ότι η υπόθεση η γωνία να περιέχεται στις δύο πλευρές είναι αναγκαία για να ισχύει το ΠΓΠ.

Ξεκινάμε κατασκευάζοντας μία κυρτή γωνία - κατά προτίμηση αμβλεία, για να είναι το σχήμα μας «άνετο», αλλά, εν γένει, δεν παίζει κάποιον ιδιαίτερο ρόλο.

Πριν κλείσουμε, να παρατηρήσουμε ότι μπορεί στη συντομογραφία ΠΓΠ να μη διατυπώνεται ρητά η αναγκαιότητα η γωνία να περιέχεται στις δύο πλευρές, ωστόσο αυτό υποδηλώνεται έμμεσα από τη σειρά παρουσίασης των στοιχείων (η γωνία αναφέρεται μεταξύ των πλευρών).
